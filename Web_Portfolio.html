Project 1 | Project Title:

Responsive E-commerce Website Development

Project 1 Details

Project Outcome:

The final product was a fully functional and responsive e-commerce website that delivers a seamless shopping experience. It achieved improved performance, enhanced usability, and mobile optimization, resulting in increased user engagement and customer satisfaction.

Key Features:

  • Responsive Design: Using HTML5, CSS3, and Tailwind CSS to create a mobile-first, fully responsive layout that adapts to various screen sizes (desktop, tablet, mobile).
  • Product Listings: Products are dynamically displayed in categories with filter and search functionalities, using JavaScript for interactive behavior and smooth UX.
  • Shopping Cart: Implemented a client-side shopping cart with real-time updates on quantity, pricing, and totals, using JavaScript.
  • User Authentication: Integrated secure user registration and login features using PHP (CodeIgniter) with hashed passwords and session management.
  • Admin Dashboard: Developed an admin panel to manage product listings, categories, users, and order processing.
  • Checkout System: Built a secure and intuitive checkout flow, including shipping details, order review, and payment integration (real API).
  • Database Integration: Used MySQL as the backend database to store user information, product details, and order history.
  • Modern UI/UX Design: Clean and engaging user interface created using Tailwind CSS with attention to color schemes, typography, and accessibility.

Technologies Used:

Frontend: HTML5, CSS3, Tailwind CSS, JavaScript, jQuery

Backend: PHP (CodeIgniter framework)

Database: MySQL

Tools: XAMPP (for local development).

Project 2 | Project Title:

Online Toys Finding Web Application

Project 2 Details

Project Outcome:

In ‚ÄúSRS‚Äù document we prepared scope of project. We also create the use case diagram demonstrate the customer of application. Functional Requirements

Admin Features:

  • Authentication: Admin can securely log in and log out of the system.
  • Category Management: Admin can add, update, and delete various categories of toys.
  • Toy Management: Admin can manage toys (name, image, price, quality, stock status).
  • Order Management: View all orders with toy details, price, user info, and shipping address.
  • Voucher Management: Generate vouchers for users during checkout.
  • Payment Verification & Order Processing: Verify payments, proceed with delivery.
  • Shipping Charges Management: Manage city-wise shipping charges auto-calculated at checkout.
  • Toy Request Feedback: View and respond to requests for unavailable toys.
  • Stock Management: Monitor and manage toy inventory levels.
  • Expense Reports: Maintain daily/weekly/monthly expense reports.
  • Profit Reports: Generate profit reports based on sales and expenses.

User/Parent Features:

  • User Registration and Login: Register and securely log in/out.
  • Profile Management: Update name, email, address, and password.
  • Toy Browsing and Search: Filter and search toys by category, price, brand, or age group.
  • Toy Information Viewing: View detailed toy info.
  • Request for Unavailable Toys: Request unavailable toys with optional image and description.
  • Shopping Cart Functionality: Add/remove toys; auto-update total price.
  • Order Confirmation: Enter address; shipping cost based on city.
  • Payment Options: Online gateway or offline payment with voucher upload.
  • Order Tracking: Track order status (Pending, Verified, Shipped, Delivered).

Non-Functional Requirements

  • Security: Secure user data, restrict access, validate inputs, and prevent attacks (SQL injection, CSRF, XSS).
  • Performance: Fast loading and responsive on slow connections/devices.
  • Reusability: Modular system components, proper backups, and validation checks.
  • Portability: Cross-platform and browser compatibility, works at various internet speeds.

Technologies Used:

Frontend: HTML5, CSS3, Tailwind CSS, JavaScript, jQuery

Backend: PHP (CodeIgniter framework)

Database: MySQL

Tools: XAMPP (for local development).
